# Guía de Responsivo

## Transferencia

### Justificación para el uso de Mobile First

Teniendo en cuenta el enfoque de la página hacia la venta de servicio de tutorías académicas, y el uso masivo de plataformas móviles, se consideró que la mayor parte del tráfico hacia la página se haría de una plataforma móvil. Considerando lo anterior, se decidió utilizar un enfoque "mobile first" al momento de realizar el código responsivo, para permitir la optimización de la página hacia dicho público, sin olvidar la posibilidad de un tráfico en la misma proveniente de dispositivos no móviles.
